# Raspberry Pi Systems Monitor

A basic web-based dashboard that displays in real-time system statistics from a Raspberry pi

## Features
- Live CPU usage
- CPU temperature
- RAM usage
- Disk usage
- Auto-refreshes every 5 seconds

## Tech Stack
- Python
- Flask
- psutil
- HTML/CSS

## How to run

1. Clone the repository
2. Install dependencies:
   pip install flask psutil
3. Run the app:
   python app.py
4. Open a browser and go to:
   http://localhost:5000

## Hardware
- Raspberry Pi 3 Model B
- Raspbian OS
